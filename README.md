Global Causes of Death — Statistical Analysis
This project analyzes a dataset containing the most common causes of death worldwide, with the goal of identifying trends and patterns across different countries and time periods.
Dataset
The dataset includes data from 195+ countries between 1990 and 2019. Each record contains the country, the year, the cause of death, and the number of deaths recorded.
Data source: Our World in Data
What the analysis covers

Which causes of death are most common globally
How mortality trends have changed over time
Differences between geographic regions and income groups

Tools used

Python, pandas, matplotlib, seaborn
Jupyter Notebook
